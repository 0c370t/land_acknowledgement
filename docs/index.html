Land Acknowledgement

Learn more about the land you are living or working on.

Just text your zip code or [city, state] to:

907-312-5085

and we'll send you the Native land you're living or recreating on.

Land Acknowledgement is a SMS bot leveraging data from Canadian Not-for-profit, Native Land, who ask that people use the data carefully because confirmation by nations is pending and they are updating data weekly.

The application is an a collaboration with the Anchorage i-team, a Municipality of Anchorage Bloomberg Philanthropy Innovation Team.

Created by Code for Anchorage.
